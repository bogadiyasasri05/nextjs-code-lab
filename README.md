# Tiny CodeLab — Internship Assignment

A small Next.js app that provides:
- Code editor (JavaScript only) using CodeMirror
- Run button — executes code in a sandboxed Web Worker
- Auto-Fix button — applies simple line-by-line fixes (semicolons, indentation, trailing spaces, bracket balancing)
- Help panel — keyword-based help responses

## Quick start

```bash
# install
npm install

# dev
npm run dev

# open
http://localhost:3000
